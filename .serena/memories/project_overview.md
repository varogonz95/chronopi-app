Purpose: Busy Time Info Device is a Raspberry Pi portrait dashboard that shows current availability, meetings, focus blocks, and upcoming events from Google Calendar, Microsoft 365, and Zoom.\nStack: Python 3, PySide6 for the dashboard UI, Flask only for setup-time OAuth callbacks, httpx for provider APIs, systemd for Pi startup.\nStructure: app/config.py loads env settings, app/providers.py manages OAuth tokens and provider fetches, app/dashboard.py builds the shared status payload, app/main.py is the Qt dashboard entrypoint, app/auth_server.py is the optional provider-setup server, deploy/ contains Pi install and launcher scripts.
